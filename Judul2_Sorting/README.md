Deskripsi Singkat
Program ini digunakan untuk mengurutkan data (tinggi badan) menggunakan algoritma Bubble Sort. Program menerima input jumlah data dan nilai tiap elemen, lalu menyimpannya dalam array. Setelah itu, data diurutkan dengan cara membandingkan elemen yang bersebelahan dan menukarnya jika tidak sesuai urutan. Terakhir, program menampilkan data sebelum dan sesudah diurutkan.
